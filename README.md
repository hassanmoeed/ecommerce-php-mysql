# E Commerce Platform

Award Winning Full Stack E Commerce Platform developed using PHP and MySQL, securing 1st Position at the University DBMS Exhibition.

## Overview

Engineered a feature rich e commerce ecosystem featuring transactional order processing, wallet based transactions, cryptocurrency enabled payment workflows, multilingual AI powered customer assistance, virtual try on capabilities, and end to end customer lifecycle management.

## Key Features

- Secure User Authentication
- Product Catalog Management
- Shopping Cart Functionality
- Transactional Order Processing
- Wallet Based Transactions
- Cryptocurrency QR Payment Infrastructure
- Return and Refund Management
- Order Tracking System
- Multilingual AI Customer Assistance
- Virtual Try On System
- Responsive User Interface

## Technology Stack

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP
- phpMyAdmin
